Synchronisation GitHub Gist

Tes données sont sauvegardées dans un Gist privé GitHub de ton compte. Aucun serveur tiers, aucun coût.
  1. Crée un PAT sur github.com/settings/tokens (Fine-grained, scope Gists: Read and write)
  2. Crée un Gist privé sur gist.github.com avec un fichier nommé atelier-data.json contenant {}
  3. Copie son ID (la longue chaîne dans l'URL) et ton PAT ici
🧪 Mode test — Bac à sable isolé. Tes données réelles restent intactes, rien n'est sauvegardé en local ni poussé vers GitHub. Chaque refresh recharge un jeu de données démo propre. Pratique pour expérimenter sans risque.
